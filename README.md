# C Template

A small C project starter using CMake and including part of [stb lib](https://github.com/nothings/stb)

## Requirements

CMake >= 3.24

## Usage

Clone

```sh
git clone tristanwagner/c-template
cd c-template
rm -rf .git
git init
```

Build

```sh
mkdir cmake
cd cmake
cmake ..
make
```

Run

```sh
./bin/c-template
```
